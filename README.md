# White Rabbit Productions — Next.js Luxury Site

This is a full starter site for the White Rabbit Productions brand system.

## Included pages

- Home
- About
- Services
- Portfolio
- Experiences
- Shop
- Contact

## Tech

- Next.js
- TypeScript
- CSS
- Framer Motion / GSAP / Lenis dependencies ready for premium animation
- Vercel-ready

## Start locally

```bash
npm install
npm run dev
```

## Deploy

Push this folder to GitHub and import the repo into Vercel.

## Important

Next.js itself does not generate images or copy by default. It is the framework. Generative features can be added by integrating:
- OpenAI API for text, concierge, recommendations, content generation
- Replicate / Runway / Stability / OpenAI image tools for image generation
- Sanity CMS for editable pages and portfolio updates

## Brand rules

- Rabbit logo is the home button
- No tagline by default
- Matte black, brushed silver, blood red
- Minimal navigation
- Cinematic page scenes
